# MindMirror — Vault của Bạn

## Đọc trước khi làm bất cứ điều gì

Đây là vault bộ não thứ hai của Tôi. Mỗi khi bắt đầu cuộc trò chuyện, đọc file [[Me]] để hiểu Tôi là ai và đang ở đâu trong cuộc đời.

## Cấu trúc Vault — MindMirror Pipeline

| Folder                  | Vai trò                                                                                                     | Quy tắc                      |
| ----------------------- | ----------------------------------------------------------------------------------------------------------- | ---------------------------- |
| `1. Thu Thập (Raw)/`    | Capture thô — ghi trước khi quên                                                                            | Không format, không phán xét |
| `2. Tinh Lọc (Mine)/`   | Khai thác — Nhật ký ngày (Nhật ký ngày (Daily Notes)), Ghi Chú Học Tập (Ghi Chú Học Tập (Literature Notes)) | Xử lý mỗi ngày               |
| `3. Chuyển Hoá (Core)/` | Hội tụ — Permanent Notes, Frameworks, Maps                                                                       | Tri thức đã tinh lọc         |
| `4. Kiến Tạo (Mint)/`   | Kiến tạo — Active, Background, Later                                                                        | Dự án và output thật         |
| `5. Công Cụ (Toolbox)/` | Công cụ — Templates, Attachments                                                                            | Không bao giờ xoá            |


## Quy trình xử lý thông tin (MindMirror Flow)

```
Input bất kỳ
    → RAW (capture thô)
    → MINE (khai thác, phân loại)
    → CORE (tinh lọc thành Permanent Notes/Frameworks)
    → MINT (biến thành output thật)
```

## Nguyên tắc làm việc với Tôi

- **Viết tiếng Việt** trừ khi Bạn dùng tiếng Anh
- **Ngắn gọn, thực chiến** — không lý thuyết dài dòng
- Khi tạo note mới, dùng template từ `5. Công Cụ (Toolbox)/Mẫu (Template)/`
- Daily Note đặt trong `2. Tinh Lọc (Mine)/Nhật ký ngày (Nhật ký ngày (Daily Notes))/YYYY-MM-DD.md`
- Wikilink `[[...]]` thay vì link Markdown khi link internal
