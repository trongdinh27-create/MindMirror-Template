# Bắt đầu từ đây — Hướng dẫn cài đặt

> Làm theo thứ tự. Tổng cộng khoảng 5 phút.

---

## Bước 1 — Cài Obsidian

1. Vào **obsidian.md** → Download → chọn Mac
2. Kéo Obsidian vào Applications
3. Mở Obsidian → **"Open folder as vault"** → chọn folder **Obsidian MindMirror Template** này
4. Obsidian hỏi cài plugins → chọn **"Trust author and enable plugins"**

---

## Bước 2 — Cài Claude Code (desktop app)

1. Vào **claude.ai/download** → tải về **Claude Code** (không phải Claude thông thường)
2. Kéo vào Applications → mở lên
3. Đăng nhập bằng tài khoản Claude.ai của bạn

---

## Bước 3 — Mở vault trong Claude Code

1. Trong Claude Code → chọn **"Open folder"**
2. Chọn folder **Obsidian MindMirror Template**
3. Xong — Claude Code đã hiểu vault của bạn

---

## Bước 4 — Khai thác bản thân lần đầu

Gõ vào chat:

```
/MindMirror-Phong-Van-Ban-Do-Cuoc-Doi
```

Claude sẽ hỏi bạn nhiều câu hỏi về bản thân và tự điền vào file [[Me]] — làm 1 lần là xong.

---

## Dùng hàng ngày

Vault này đi kèm sẵn **1 skill**: `/MindMirror-Phong-Van-Ban-Do-Cuoc-Doi` — phỏng vấn sâu để xây Hồ sơ Bản thân (đã chạy ở Bước 4).

Ngoài ra, cứ **nói chuyện trực tiếp** với Claude Code — nó đã hiểu cấu trúc vault (RAW → MINE → CORE → MINT) và giúp bạn ghi chép, chắt lọc, tổng hợp mà không cần lệnh riêng. Ví dụ:

- *"Lưu ý tưởng này vào Thu Thập giúp tôi"*
- *"Chắt lọc đoạn này thành ghi chú trong Core"*
- *"Tổng hợp lại tuần này cho tôi"*
